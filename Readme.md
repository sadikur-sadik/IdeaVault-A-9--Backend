# Idea Vault Project Backend

## Description :
### This backend is created using Express JS (Node JS), and Mongo DB as database.This backend handles users request, response, and also store there information inside the database and returns an API

## CRUD Operations : 

- Post Ideas (Secured with JWT)
- Patch Ideas (Secured with JWT)
- Delete Ideas (Secured with JWT)
- Get Ideas
- Getting Featured Ideas using limit method
- Post Comments (Secured with JWT)
- Patch Comments (Secured with JWT)
- Delete Comments (Secured with JWT)
